# Final Project

Meet Wayan
a Barong by Adriella Kristie

A Barong is a Balinese mythological creature known to be a sacred guardian spirit. I wanted to make a fun interactive animation that also introduces people to the Balinese folklore character Barong. I made it a character with a little name and provided a bit of an activity to make it more fun!
Hope you enjoy meeting Wayan, playing with the kaleidoscope, and getting your sticker at the end! :))

ps. make sure you have your volume up!
